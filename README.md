# Inkwell – Full Stack Blog Application

## Overview

Inkwell is a modern full-stack blogging platform built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). It provides a clean and distraction-free environment where users can create, publish, and interact with blog content.

The platform allows users to register, authenticate securely, write rich-text articles, upload cover images, like posts, and participate in discussions through comments. Inkwell aims to provide a simple yet powerful blogging experience for writers and readers alike.

## Key Features

### User Authentication

* User Registration
* User Login
* JWT-Based Authentication
* Protected Routes

### Blogging Features

* Create Blog Posts
* Edit Existing Posts
* Delete Posts
* View All Posts
* Rich Text Editor (React Quill)

### Media Support

* Upload Cover Images
* Cloudinary Integration
* Image Display within Posts

### Community Interaction

* Like / Unlike Posts
* Comment on Posts
* View Comments in Real Time

### User Interface

* Modern Landing Page
* Responsive Navigation
* Dashboard
* Blog Feed

## Technology Stack

### Frontend

### Backend

* Node.js

### Authentication

### Cloud Services

## Project Goal

The goal of Inkwell is to provide a complete blogging platform where users can publish ideas, share stories, and engage with content through likes and comments while enjoying a clean and modern writing experience.
